## What is Salesforce Platform?

Salesforce Platform is a cloud-based platform used to build and manage business applications. It provides tools for managing data, creating apps, automation and customization without needing much hardware or setup.

---

## Salesforce Concepts

### App
An App in Salesforce is a collection of related objects, tabs and features designed for a specific business purpose.

Example:
Sales App, Service App or College Management App.

---

### Object
An Object is used to store data in Salesforce.

It is similar to a table in a database.

Examples:
- Student
- Faculty
- Course
- Account
- Contact

---

### Tab
A Tab is used to access objects, records or features in Salesforce through the user interface.

Example:
Student Tab or Course Tab.

---

## Difference Between App and Object

An App is a collection of multiple objects and features, while an Object is used to store specific data.

For example:
A College Management App can contain Student, Faculty and Course objects.

---

## Configuration vs Coding

### Configuration (No Code)
Configuration means customizing Salesforce using clicks and built-in tools without writing code.

Examples:
- Creating objects and fields
- Building flows and reports

### Coding (Apex)
Coding is used when advanced customization or business logic is needed.

Examples:
- Custom validation logic
- Complex automation using Apex

---

## Multi-Tenant Architecture

Multi-tenant architecture means multiple companies use the same Salesforce platform and resources securely while keeping their data separate.

---

## How Developers Extend Salesforce

Developers can extend Salesforce functionality using:
- Apex
- APIs
- Lightning Web Components
- Integrations
- Automation tools
